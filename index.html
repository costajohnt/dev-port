Prophecy Sciences

Python and js

Prophecy Sciences is a Y-Combinator backed startup with neuroscience-based games that provide insights on personality, aptitude, and optimal career fits. We were responsible for full stack development (including backend database design, and front end javascript game-play) as well as deployment and maintenance of the application on Heroku.

Additionally, we integrated with their development team as they hired new employees, and helped with setup and pair programming to bring new devs up to speed.

prophecysciences.com

Nouvola

Rails Full Stack

Nouvola is a cloud based, stress testing service, which can mimic hundreds of thousands of users visiting a website.

We were responsible for implementing a front end redesign after a complete UI overhaul. Heavy javascript and ajax were required, to create a seamless user experience for a complicated back end infrastructure.

nouvola.com
